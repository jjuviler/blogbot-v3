Modify source code:

ⓘBlogbot adds the "more" tag after the first paragraph in the body text. If no paragraph is found, a "more" tag will not be placed.
ⓘBlogbot clears all span tags from the document. Note that if you are using codepen, the embed code requires span tags.
ⓘBlogbot clears all style attributes from all tags (except for table tags) in the document.
ⓘBlogbot deletes empty lines to reduce extra space between paragraphs.
ⓘBlogbot replaces all instances of the special character nbsp (non-breaking space) with a normal space.
ⓘBlogbot makes all links in the body text (besides anchor links) to open in a new tab, and also adds the rel="noopener" attribute to these links.
ⓘBlogbot converts straight quotes to smart (aka curly) quotes.
ⓘBlogbot applies formatting to all images in the post body: centered, 650px wide.
ⓘBlogbot applies formatting to all "Image Source" links: centered, 12px, italicized, both words capitalized.
ⓘIf there is alt text written under an image (that begins with "alt:" or "alt text", case insensitive), Blogbot removes this text from the post body and makes it the alt text of the preceeding image.
ⓘIf there is an image name written underneath an image (starting with "IMG name:" or "Image name:", case-insensitive), Blogbot removes this line of text from the post body.
ⓘBlogbot adds anchors above all H2s and featured snippets. These anchors can be linked to in a table of contents. This option is turned off by default.

Check for formatting/style issues:

ⓘBlogbot checks for skipped headings (e.g., an h4 following an h2).
ⓘBlogbot checks for instances of double-hyphens (--), which should be replaced with em dashes.
ⓘBlogbot checks that ellipses have spaces between surrounding text.
ⓘBlogbot checks that "vs." is used only in headings and "versus" is used only in paragraphs.
ⓘBlogbot checks for tracking parameters in all URLs (with the exception of YouTube URLs) by scanning for the "?" character. If a "?" is found in a URL, Blogbot flags it. Everything including and after the "?" in the URL should be removed as long as the link still works properly without it.

Featured snippets:

See full instructions here. ⓘCreates paragraph/list snippets from body copy. To create a featured snippet, place your featured snippet content between [FS] and [/fs] tags.
